What is Turkcoin?
----------------

Turkcoin is a lite version of Bitcoin using scrypt as a proof-of-work algorithm.
 - 5 minute block targets
 - ~350k total coins

The rest;
 - 20 coins per block
 - 2016 blocks to retarget difficulty
